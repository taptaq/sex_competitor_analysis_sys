# IntimateIntel | 竞品分析与创业压力测试 SaaS

专为情趣用品行业经理人设计的专业级 SaaS 平台。以“大人糖”等行业巨头为基准，提供真实、客观且硬核的市场洞察。

## 🚀 核心功能

- **数据仪表盘**: 实时可视化市场机会、价格趋势和用户情感雷达图。
- **竞品深度扫描**: 以“大人糖”为例，深度解析其产品策略、宣传语创意及用户负面预警。
- **创业压力测试**: 注入 Gemini 3 强效 AI，模拟最真实的市场环境，提供“解毒式”创业建议。

## 🛠 技术栈

- **前端**: React 18, TypeScript, Tailwind CSS, Vite.
- **状态管理**: Zustand.
- **图表库**: Recharts.
- **AI 引擎**: Google Gemini API.

## 📦 如何本地运行

1. 安装依赖:
   ```bash
   npm install
   ```
2. 启动开发服务器:
   ```bash
   npm run dev
   ```

_注意：请确保在环境变量中配置 `API_KEY` 以激活 AI 模块。_
